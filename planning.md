# Planejamento

## Função `f1()`

**Entrada:**

```csv
nome;idade
André;22
Sancler;46
```

**Saída:**
```json
[
    {
        "nome": "André",
        "idade": "22",
    },
    {
        "nome": "Sancler",
        "idade": "46",
    },
]
```


## Função `printTabela()`

**Entrada:**
```json
[
    {
        "nome": "André",
        "idade": "22",
    },
    {
        "nome": "Sancler",
        "idade": "46",
    },
]
```

**Saída:**
|nome|idade|

|Andre|22|
|Sancler|46|
